# LocalOJ

## 简介

用于本地评测程序，使用C++17标准，只适用于Windows平台！

用户指定存放测试集的文件夹或者指定一组测试用例的文件，再指定要评测的exe文件

本工具可完成程序在本地的评测，并且对于WA的样例，高亮显示差异
